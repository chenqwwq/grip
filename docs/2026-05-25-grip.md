# Grip 当前功能整理

更新时间：2026-05-27

Grip 是一个 macOS 菜单栏待办工具。核心目标是把屏幕截图、剪贴板文字或剪贴板图片快速识别成待办任务，并支持在本地任务列表和系统 Reminders 之间同步。

## 任务创建

Grip 支持三种任务来源：

1. 区域截图创建
   - 快捷键：`Command + Shift + T`
   - 菜单栏入口：`区域截图创建`
   - 应用内使用 ScreenCaptureKit 获取屏幕内容，并显示自定义选区遮罩。
   - 选区截图会提交给图片模型识别。

2. 剪贴板创建
   - 快捷键：`Command + Shift + V`
   - 菜单栏入口：`从剪贴板创建`
   - 优先读取剪贴板文字。
   - 如果没有文字，会读取剪贴板图片。
   - 文字提交给文本模型，图片提交给图片模型。

3. 手动任务
   - 数据模型已支持 `manual` 来源。
   - 当前主要入口仍是识别后确认创建。

## 识别结果确认

识别成功后不会直接创建任务，而是先弹出“确认新任务”面板。

面板可编辑：

- 标题
- 详细描述
- 分类
- 优先级
- 截止日期
- 原始截图或原始文本预览

点击“创建”后才会写入本地任务库。标题为空时无法创建。

## 任务列表

主窗口展示所选日期的任务列表。

顶部工具栏包含：

- 标题：`今日待办`
- 日期点选器
  - 显示“今天 / 明天 / 昨天 / M月d日”
  - 点击后弹出图形日历
  - 左右箭头可切换前一天、后一天
- 筛选器
  - 全部
  - 待处理
  - 已完成

任务行能力：

- 左侧圆圈可直接完成或恢复待处理。
- 除圆圈外，整行可点击打开任务详情。
- 行内展示标题、分类、优先级、截止时间。
- 右侧展示任务摘要：
  - 优先展示详细描述
  - 没有详细描述时展示原始文本、截止时间或来源说明
  - 展示来源、同步状态、创建时间
- 行分隔线避开左侧完成按钮，不会穿过按钮区域。

## 任务详情

点击任务行后打开任务详情弹窗。

可编辑：

- 标题
- 详细描述
- 分类
- 优先级
- 截止日期

可查看：

- 原始截图
- 原始文本
- 创建日期
- Reminders 同步状态

可操作：

- 保存
- 删除
- 取消

详细描述输入框已增加内部留白，避免文字被边框或标题遮挡。

## 菜单栏

Grip 在系统菜单栏提供状态栏入口。

菜单内容：

- 今日任务列表
  - 最多展示 10 个任务
  - 点击任务可完成或恢复待处理
  - 鼠标悬浮展示任务详情 tooltip
- 打开任务详情子菜单
  - 点击任务打开详情弹窗
  - 鼠标悬浮同样展示详情
- 区域截图创建
- 从剪贴板创建
- 打开主窗口
- 退出 Grip

## Reminders 同步

支持同步到 macOS 系统 Reminders。

同步模式：

- 自动同步
- 手动同步
- 关闭同步

手动同步入口：

- 主窗口底部 `同步到 Reminders`

同步内容：

- 标题
- 详细描述
- 优先级
- 截止日期
- 完成状态

双向完成状态同步：

- 设置里可开启或关闭。
- 开启后，同步时会从 Reminders 拉取完成或待处理状态。
- 本地完成或恢复待处理后，如果任务已同步或同步模式为自动，会推送到 Reminders。
- 如果 Reminders 中的任务被删除，Grip 不删除本地任务，只清除同步关联。

## 设置

设置页包含：

- 文本模型配置
- 图片模型配置
- API URL
- 模型名称
- API Key
- Reminders 同步模式
- Reminders 完成状态双向同步开关
- 日志开关
- 日志文件路径

API Key 存储在 Keychain 中，不写入 UserDefaults。

## 权限策略

当前涉及的系统权限：

1. 录屏权限
   - 必要权限。
   - 用于区域截图。
   - 应用启动时统一检查并申请。
   - 未授权时显示阻塞提示，提供“打开系统设置”和“重新检查”。

2. Reminders 权限
   - 非必要权限。
   - 用于同步系统提醒事项。
   - 首次打开应用时申请一次。
   - 后续不会在每次启动时重复申请；用户触发同步时会按需检查。

剪贴板读取不属于预授权式权限，不在启动时申请。

## 日志

Grip 内置日志能力。

日志输出位置：

- 系统日志
- 控制台输出
- 文件日志

日志覆盖的关键流程：

- 应用启动
- 权限检查
- 剪贴板读取
- 区域截图
- LLM 请求和响应
- 识别成功或失败
- 任务创建、更新、完成
- Reminders 同步
- Keychain 和配置相关错误

## 快捷键

- `Command + Shift + T`：区域截图创建任务
- `Command + Shift + V`：从剪贴板创建任务
