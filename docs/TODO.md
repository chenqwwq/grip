# Grip TODO

更新时间：2026-05-27

## 高优先级

- [ ] 自然语言快速新建
  - 在主窗口或菜单栏提供快速输入入口。
  - 输入示例：`明天下午三点提醒我交周报`。
  - 复用现有文本模型解析流程，生成 `TaskDraft` 后进入确认面板。

- [ ] 任务延期 / 稍后提醒
  - 在任务行或详情页提供快捷延期操作。
  - 预设选项：今天晚点、明天、下周一、自定义。
  - 延期后更新任务 `dueDate`，并在已同步时同步到 Reminders。

## 中优先级

- [ ] 今日概览统计
  - 在主窗口顶部展示今日总数、待处理数、已完成数、逾期数。
  - 日期切换和筛选变化后实时刷新。

- [ ] 菜单栏迷你输入
  - 在状态栏菜单增加“快速输入...”入口。
  - 打开轻量输入面板，输入一句话后创建任务草稿。
  - 与自然语言快速新建共用解析和确认流程。

- [ ] 来源回看增强
  - 截图任务详情支持放大查看原截图。
  - 剪贴板文本任务支持一键复制原文。
  - 剪贴板图片任务支持查看原图。

## 低优先级

- [ ] 智能去重
  - 创建任务前检测相似标题、相同截止时间或相同来源内容。
  - 如果疑似重复，在确认面板提示可能已有类似任务。
  - 用户仍可选择继续创建。
